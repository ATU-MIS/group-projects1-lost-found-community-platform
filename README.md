# Lost & Found Community Platform

Lost & Found Community is a community-driven web platform designed to help people report lost items, list found items, and reconnect belongings with their rightful owners. The platform promotes collaboration, social responsibility, and trust within a shared digital community.

This project was developed following **Software Engineering principles** and supported by UML diagrams including **Use Case**, **Class**, and **Sequence** diagrams.
